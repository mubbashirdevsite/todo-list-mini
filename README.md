# 📝 Modern To-Do List App

A modern, responsive To-Do List application built using **HTML, CSS, and Vanilla JavaScript** with full CRUD functionality and Local Storage support.

---

## 🚀 Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Live search functionality
- Filters (All / Active / Completed)
- Task statistics (Total / Completed / Pending)
- Local Storage persistence
- Clear completed tasks
- Current date display
- Empty state message
- Smooth animations
- Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
